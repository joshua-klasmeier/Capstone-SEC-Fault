# Automated Finance Reporting Tool

## Overview
This project is an Automated Finance Reporting Tool that takes in SEC filings and generates simple, easy-to-read summaries based on user prompts.

The goal is to help users stay informed about company financial updates without needing a strong background in finance or accounting.

## Features
- Fetches SEC filings using the SEC EDGAR API
- Generates plain English, news-style summaries
- Uses LLM-powered summarization
- Designed for users who want quick financial insights

## Tech Stack
**Frontend:** Next.js, React, TypeScript  
**Backend:** Python (FastAPI)  
**Database:** PostgreSQL, Qdrant (Vector DB for RAG), Redis  
**LLM Provider:** Gemini
